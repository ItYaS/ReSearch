Документация ReSearch

ReSearch - это python-репозиторий с открытым исходным кодом для извлечения данных из файлов и веб-страниц.

Он простой, понятный и для его использования много не нужно.

Что бы начать пользоваться репозиторием, для начала установите python c его официальной страницы. После установите пакет через консоль.

> pip install research

Затем необходимо импортировать библиотеку в вашем скрипте:

import research as rs

Репозиторий делится на части связанные с форматами и операциями. Вот так выглядит общий вызов функций:

rs.формат.метод(аргументы)

Парсинг данных из форматов

Формат .txt

Пример вызова функции для этого формата:

rs.txt_format.функция(аргуметы)

Возможные методы:

get_text(path, coding='utf-8') - функция, которая возвращает текст из файла, в виде списка, где элементы списка - это предложения в файле.

path - путь к нужному файлу.

coding - кодировка для декодирования. По умолчанию стоит кодировка utf-8.


find_text(path, pattern_list, coding='utf-8') - функция, которая возвращает совпадения из текста файла в виде словаря, где ключ - это совпадение, а значение - это шаблон поиска по которому было найдено совпадение.

path - путь до нужного файла.

pattern_list - список с шаблонами, где каждый шаблон является отдельным элементом.

coding - кодировка для декодирования. По умолчанию стоит кодировка utf-8.

Формат .rtf

Пример вызова функции для этого формата:

rs.rtf_format.функция(аргуметы)

Возможные методы:

get_text(path, coding='utf-8') - функция, которая возвращает два списка, первый - текст из файла, где элементы списка - это предложения в файле, а второй - стили текста.

path - путь к нужному файлу.

coding - кодировка для декодирования. По умолчанию стоит кодировка utf-8.


find_text(path, pattern_list, coding='utf-8') - функция, которая возвращает совпадения из текста файла в виде словаря, где ключ - это совпадение, а значение - это шаблон поиска по которому было найдено совпадение.

path - путь до нужного файла.

pattern_list - список с шаблонами, где каждый шаблон является отдельным элементом.

coding - кодировка для декодирования. По умолчанию стоит кодировка utf-8.

Формат .csv

Пример вызова функции для этого формата:

rs.csv_format.функция(аргуметы)

Возможные методы:

get_text(path, coding='utf-8') - функция, которая возвращает текст из файла, в виде списка, где каждый вложенный список - это строка, а элементы вложенного списка - это столбцы в строке.

path - путь к нужному файлу.

coding - кодировка для декодирования. По умолчанию стоит кодировка utf-8.


find_text(path, pattern_list, coding='utf-8') - функция, которая возвращает совпадения из текста файла в виде словаря, где ключ - это совпадающая строка, а значение - это шаблон поиска по которому было найдено совпадение.

path - путь до нужного файла.

pattern_list - список с шаблонами, где каждый шаблон является отдельным элементом.

coding - кодировка для декодирования. По умолчанию стоит кодировка utf-8.

Парсинг текста

Пример вызова функции для этого формата:

rs.parsing.функция(аргуметы)

Возможные методы:

get_text(url) - функция, которая возвращает текст из страницы, в виде списка, где элементы списка - это информация из тегов(h, p, a, img).

url - адрес нужной страницы.


find_text(url, pattern_list) - функция, которая возвращает совпадения из тегов страницы в виде словаря, где ключ - это совпадение, а значение - это шаблон поиска по которому было найдено совпадение или ссылка, содержащаяся в теге.

url - адрес нужной страницы.

pattern_list - список с шаблонами, где каждый шаблон является отдельным элементом.
