Формат .json

Вызов функции этого формата:

research.json_format.функция(аргуметы)

Возможные методы:

get_data(path) - функция, которая возвращает данные из файла, в виде списка, где каждый вложенный список - это строка, а элементы вложенного списка - это столбцы в строке.

path - путь к нужному файлу.


find_data(path, pattern_list) - функция, которая возвращает совпадения из данных файла в виде словаря, где ключ - это шаблон поиска, а значение - это строка с найденным совпадением.

path - путь до нужного файла.

pattern_list - список с шаблонами, где каждый шаблон является отдельным элементом.


write_data(path, data, mode='w') - функция, которая записывает данные в файл и сохраняет его.

path - путь файла в который надо записать данные.

data - словарь с данными, которые нужно записать в файл.

mode - режим открытия файла. По умолчанию стоит режим чтения 'w'.


replace_data(path, new_path, data_dict, mode='w') - функция, которая принимает словарь с данными, заменяет данные и сохраняет эти изменения в новом файле.

path - путь файла из которого надо взять данные.

new_path - путь файла, в который надо записать изменения. Если ввести тот же путь, то изменения сохранятся в этом файле.

data_dict - словарь со значениями, где ключи словаря - это значение, которое надо заменить, а значение элемента словаря - значенин на которое надо заменить.

mode - режим открытия файла. По умолчанию стоит режим чтения 'w'.
