Документация ReSearch

ReSearch - это python-репозиторий с открытым исходным кодом для извлечения данных из файлов и веб-страниц.

Он простой, понятный и для его использования много не нужно.

Что бы начать пользоваться репозиторием, для начала установите python c его официальной страницы. После установите пакет через консоль.

> pip install research

Репозиторий делится на части связанные с форматами и операциями. Вот так выглядит общий вызов функций:

research.формат.метод(аргументы)

В меню вы найдете возможные форматы и операции.
